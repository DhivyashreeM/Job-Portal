# Job Portal

## Overview

Job Portal is a full-stack web application designed to simplify the recruitment process by connecting job seekers with recruiters. The platform provides a modern, secure, and user-friendly environment where recruiters can post job openings and manage applicants, while job seekers can create professional profiles, search for jobs, and apply online.

The application is built using a **React + TypeScript** frontend and a **Spring Boot** backend, following a RESTful architecture. It uses **JWT (JSON Web Token)** for secure authentication and authorization, ensuring that only authorized users can access protected resources.

## Features

### Job Seeker

* Secure user registration and login
* JWT-based authentication
* Create and update professional profile
* Add educational qualifications, work experience, and certifications
* Browse available job openings
* Search and filter jobs
* Apply for jobs
* Track application status
* Receive notifications regarding applications

### Recruiter

* Secure login and authentication
* Post new job openings
* Update existing job listings
* Manage job postings
* View applicants for each job
* Review applicant profiles
* Update application status
* Send notifications to applicants

## Application Workflow

1. Users register as either a Job Seeker or Recruiter.
2. Users log in securely using JWT authentication.
3. Recruiters create and publish job postings.
4. Job seekers browse available jobs.
5. Applicants submit applications for suitable positions.
6. Recruiters review applications and update their status.
7. Notifications are sent to users regarding important updates.
